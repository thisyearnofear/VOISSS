# Base Sepolia Deployments

## Web App Contracts

### BaseVoiceStorage
- **Function**: Decentralized voice recording storage and metadata management
- **Integration**: Core storage for web app recordings

### BaseUserRegistry
- **Function**: User registration, profiles, and social features
- **Integration**: Identity management for web app users

### BaseAccessControl
- **Function**: Privacy controls and access permissions for recordings
- **Integration**: Access management for web app content

## Network Information

- **Network**: Base Sepolia Testnet
- **RPC**: https://sepolia.base.org
- **Explorer**: https://sepolia.basescan.org

## Web App Features

These contracts enable:
- Gasless transactions via Base Account SDK
- Decentralized storage indexing
- User identity and social features
- Content access control
